Chipolette is a tiny CSS framework / Starter kit.

It’s a fork from Shoelace and Bootstrap, that fully embraces CSS variables / custom properties.

Motivation Installation Text inputs Checkboxes Radios Buttons Navbars Alerts

Motivation

No compilation required to change colors / sizes etc. Well that is if you don’t want to compile. It’s still written in LESS, because of nesting and other neat features that eases development of Chipolette.

It is designed to replace Bootstrap and to be used with CSS variables.

We can just defined variables in runtime or set prefedined ones, to change whole color scheme (buttons, links, background, margins etc.)

/* Just change every available variable to your disire */
:root {
    --state-primary: #ef7310;
    --state-warning: #ffc31b;
    ...
    --body-color: #6d5346;
    --body-bg-color: #eee6e1;
    --font-system: 'Open Sans';
    --headings-font-family: 'Bitter';
    --headings-margin-bottom: var(--margin-md);
}

Installation

To install the stable version:

npm install --save chipolette

and just import LESS version like so

@import '../node_modules/chipolette/src/chipolette.less';
// or CSS version
@import '../node_modules/chipolette/dist/chipolette.min.css';

This assumes you are using npm as your package manager.

If you’re not, you can access it on unpkg like so

<link rel="stylesheet" href="https://unpkg.com/chipolette@latest/dist/chipolette.css" />

Browser Compatibility

Every evergreen browser and older ones that support CSS variables. But for older IE versions (and other outdated browsers) there is a polyfill.

Text input variables

Variable name Description
--input-height Used for input as height
(defaults to 2.4rem)
--input-height-xs|sm|lg|xl (defaults to 1.35rem|2rem|3rem|3.35rem)
--input-font-size Used for input as font size
(defaults to 1rem)
--input-font-size-xs|sm|lg|xl (defaults to .625rem|.875rem|1.25rem|1.5rem)
--input-font-family Used for input as font size
(defaults to inherit)
--input-font-weight Used for input as font weight
(defaults to inherit)
--input-border-radius Used for input as border radius
(defaults to --component-border-radius)
--input-border-width Used for input as border width
(defaults to --component-border-width)
--input-border-color Used for input as border color
(defaults to --component-border-color)
--input-speed Used for input as animation/transition speed for background, color and border
(defaults to --component-speed)
--input-loader-size Used for input as spinner diameter
(defaults to 1em)
--label-color Used for label as color
(defaults to inherit)
